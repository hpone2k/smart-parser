Project Overview

The Micron Smart Tool Log Parser is an AI-powered solution built for the AISG National AI Student Challenge by Team Algorithm. It provides a unified intelligence layer to understand diverse semiconductor tool logs across different machine types and vendors.


Traditional rule-based parsers break when firmware updates. This system uses Claude AI to intelligently extract structured insights from any log format and normalizes them into a unified database schema ready for analytics.
